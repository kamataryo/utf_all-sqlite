# utf_all-sqlite

郵便番号データの Sqlite を作成するツール。

## 前提

- Node.js@18
- SQLite3

## 使い方

```shell
$ git clone git@github.com:kamataryo/utf_all-sqlite.git
$ cd utf_all-sqlite
$ yarn
$ npx ts-node src/main.ts # ダウンロード・SQLite ファイル作成
$ sqlite3  ~/.utf_all-sqlite/data.sqlite 
$ sqlite3 ~/.utf_all-sqlite/data.sqlite ".headers on ; select * from utf_all limit 10"
```
